The Item Request System is a specialized enterprise web application engineered to digitize and optimize the supply requisition process at La Rose Noire Philippines. By transitioning from a manual, paper-based workflow to this centralized digital platform, the organization ensures accountability, inventory control, and operational efficiency.

Central to the system is a sophisticated logic engine that enforces Uniform & PPE Allowances based on employee department and hierarchy, ensuring compliance with company allocation policies (daily vs. deployment limits) before a request is even submitted.

1. Intelligent Allowance Engine

The system parses a complex matrix of allowance rules (CSV/Database) to dynamically validate requests. It distinguishes between Daily Consumables (Safety items) and Deployment Assets (Uniforms), preventing over-requisition based on department/rank entitlement.

2. Dynamic Approval Routing

Requests are automatically routed to the correct approver based on the requestor's department. The system recognizes primary and secondary approvers, adhering to the organization's escalating hierarchy (Staff ‚Üí Supervisor ‚Üí Manager).

3. Role-Based Access Control (RBAC)

Strict security protocols differentiate user experiences:

  • Staff: Can only initiate requests and view their own history. Permissions are validated via session tokens.
  • Approvers: Granted specific action capabilities (Approve/Reject) only for requests matching their managed department.
  • Admin: Full CRUD access to the Item Master and global view references.

4. Admin Item Management

A dedicated portal allows Admins to maintain the Item Master List. Features include category segmentation, pricing updates, and UOM configuration, ensuring the catalog remains synchronized with actual inventory.

5. Reporting & PDF Generation

Utilizing a print-optimized CSS engine, the system generates standardized PDF documents for individual requests or batch summaries, facilitating offline verification and audit trails.
